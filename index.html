How to Use the Dashboard

This dashboard is designed for simplicity and efficiency.
  • Home: Get a quick overview of key metrics and recent activities.
  • Profile Management: View, edit, or delete student profiles. Status icons indicate if a student is active, pending, or inactive.
  • Registration: Register new students by filling out the form. Select a course from the dropdown menu.
  • Dark/Light Mode: Use the moon/sun icon in the header to toggle between themes.
  • Sidebar: The sidebar can be collapsed on desktop for more screen space and is accessible via the hamburger menu on mobile.
Always double-check information before submitting or deleting records.
